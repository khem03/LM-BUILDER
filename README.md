# Link Matrix Builder

A simple, user-friendly tool for generating attribute matricesâ€”perfect for rapidly creating lists of HTML link attributes, database seed data, or any scenario where you need to combine multiple columns of values into a structured output.

## Features

- **Add unlimited columns:** Define any number of attribute columns (e.g., `href`, `text`, `target`, etc.).
- **Batch input:** Paste or type multiple values (one per line) per column.
- **Instant output:** Generate attribute string combinations for each row.
- **Copy to clipboard:** One-click to copy the generated output.
- **Responsive UI:** Clean, mobile-friendly design.
- **No dependencies:** Pure HTML, CSS, and JavaScript.

## Usage

1. **Clone or download** this repository.
2. Open `index.html` in your web browser.
3. For each column:
    - Enter a variable/attribute name (e.g., `href`, `text`, etc.).
    - Enter values, one per line, in the textarea below the name.
4. Click **+ Add Column** to add more columns as needed.
5. Click **Generate** to produce the output.
6. Copy the output using the **Copy Output** button.
